**a标签的用法:** 跳转外部页面、跳转内部锚点、跳转到邮箱或电话等  
```html
<a href="www.baidu.com">百度</a>
<a href="mailto:313405510@qq.com">邮箱</a>
<a href="tel:13479097715">手机号</a>
```  
**img标签的用法**:发出get请求,展示一张图片  
```
<img width="400" src="https://www.img.xz95.top/img/wikipedia/zh/9/98/KamenRiderKuuga-title.jpg" alt="">图片
```
**table标签的用法**:用来展示数据
```
    <table>
        <thead>
            <tr>
                <th></th>
                <th>小红</th>
                <th>小明</th>
                <th>小颖</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <th>数学</th>
                <td>61</td>
                <td>91</td>
                <td>85</td>
            </tr>
        </tbody>
        <tfoot>
            <tr>
                <th>总分</th>
                <td>200</td>
                <td>200</td>
                <td>200</td>
            </tr>
        </tfoot>
    </table>
```
